## ✨ Features
- **Modern Glassmorphism UI:** A sleek, frosted-glass design with mesh gradient backgrounds.
- **Real-Time History:** Tracks and stores your calculations using Browser LocalStorage.
- **Responsive Design:** Optimized for both desktop and mobile devices.
- **Keyboard Support:** Full integration for physical keyboard inputs (numbers, operators, and shortcuts).
- **Advanced Logic:** Handles complex expressions including bracketed multiplication automatically.

---

## 🛠 Tech Stack
- **Frontend:** HTML5, JavaScript (ES6+)
- **Styling:** CSS3 (CSS Grid, Flexbox, Backdrop-filter)
- **Typography:** Inter & Fjalla One (Google Fonts)
- **Storage:** LocalStorage API
